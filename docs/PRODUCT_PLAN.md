# 모바일 청첩장 서비스 기획

## 목표

디자인 경험이 없어도 예비 신랑, 신부가 10분 안에 예쁜 모바일 청첩장을 만들고 공유할 수 있게 한다.

## 핵심 사용자

- 제작자: 청첩장을 직접 만들고 사진, 지도, 계좌, 방명록을 관리한다.
- 하객: 링크로 접속해 예식 정보를 확인하고 메시지 작성, 계좌 복사, 지도 열기, 참석 여부 전달을 한다.

## 제작자 플로우

1. 간편 로그인
2. 새 청첩장 만들기
3. 템플릿 선택
4. 기본 정보 입력
5. 섹션 구성 선택
6. 사진과 음악 등록
7. 모바일 미리보기
8. 링크 생성
9. 카카오톡, 문자, 링크 공유

## MVP

- 로그인
- 청첩장 생성, 수정, 삭제
- 템플릿 3종 이상
- 기본 정보 입력
- 이미지 업로드와 갤러리
- 지도와 길찾기
- 계좌번호 등록과 복사
- 방명록 작성, 수정, 삭제
- 공유 링크 생성
- 공개 청첩장 페이지

## Beta

- RSVP
- 음악 등록
- QR 코드
- 방문 통계
- 섹션 순서 변경
- 카카오톡 공유 최적화

## 정식 출시

- 결제
- 프리미엄 템플릿
- 관리자 페이지
- 커스텀 URL
- 방문 통계 고도화

## 차별화 포인트

- 노션처럼 자유로운 섹션 구성
- 모바일 실시간 미리보기
- 하객용 UX 최적화
- 예식장 주소 기반 오시는 길 자동 추천
- 감성 템플릿과 섬세한 애니메이션
